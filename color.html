Color Theory:

Color theory is the study of how colors interact, how they can be combined, and how they influence perception and emotion. It provides a framework for understanding color relationships and is widely used in art, design, and visual communication.

Background properties:

CSS backgrounds properties are used to control the background effects of elements.

  • background-color:
  • background-image:
  • background-repeat:
  • background-position:
  • background-size:
  • background-attachment:
  • background-clip:
  • background: 
background-color: sets the background color of an element

background-image: sets the images as the background 

background-repeat: controls how the image is repeated 
- repeat-x - only horizontal 
- repeat-y - only vertical 
- no-repeat - only one image

background-position: sets the starting position of the background image 
can use keywords: top, bottom, left, right, centre or  
values:50px 50px; 

background-size: define the background size of an image 
- auto - original size(default) 
- cover - fill entire area 
- contain -- fit inside area 
or set exact size - 200px 100px 

background-attachment: Determines if background scrolls with the page or stays fixed 
- scroll - moves with page(default) 
- fixed - stays in place when scrolling

background-clip: defines how far background extends
