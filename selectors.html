CSS Selectors:

Selector is nothing but select the element which you wanted to style. It tells the browser Which element should this style apply to?

CSS Syntax: 
selector { 
    property: value; 
} 


selector ‚Üí identifies the HTML element(s). 
property ‚Üí the style property (like color, font-size, etc.). 
value ‚Üí the value you assign to that property.

Types of selectors:

  1. Basic/Simple selectors
  2. Combination selectors
  3. Pseudo class selectors
  4. Pseudo element selectors

Types of Basic/Simple Selectors:

1.Element / Tag Selector :

‚Ä¢ Selects all elements of a specific type (tag name) 

Syntax:

    property: value; 
}

2. Class Selector : 

‚Ä¢ Selects elements with a specific class attribute. 
Symbol : .(dot) 
Syntax:

    property: value; 
}

3.ID Selector : 

‚Ä¢ Selects an element with a unique id attribute. 
Symbol: #(hashtag) 
Syntax:

    property: value; 
}

4.Group Selector : 

‚Ä¢ Allows styling multiple selectors at once by separating them with commas. 
Symbol: ,(comma) 
Syntax: 
selector1, selector2, selector3 { 
    property: value; 
}

5.Universal / Global Selector : 

‚Ä¢ Selects all elements on the page. 
Symbol: * (astrick)
Syntax:

    property: value; 
}

Combination Selectors:(Combinators) :

In Css, between two simple selectors we can place a combinator this is called combination selectors

Types of Combinators:

  • Descendant selector
  • Child selector
  • Adjacent sibling selector
  • General sibling selector

Descendant Selector (space): 

Select all direct and indirect children for styling 
Syntax: 
parent descendant { 
   property: value; 
}

Child Selector (>) :

Selects only the direct children of a parent, not deeper levels. 
Syntax: 
parent > child { 
   property: value; 
}

Adjacent Sibling Selector (+) :

Selects the immediately next element after a given element 
Syntax: 
element1 + element2 { 
   property: value; 
}

General Sibling Selector (~) : 

Selects all siblings (on the same parent level) that come after a 
given element. 
Syntax: 
element1 ~ element2 { 
   property: value; 
}

Pseudo selectors:

Pseudo-selectors in CSS are special keywords added to selectors that let you style elements based on their state, position, or part without needing extra classes or IDs. They start with a colon : (or) double colon :: for pseudo elements.

Types of Pseudo selectors: 

Mainly there are two types of pseudo selectors there 
1.Pseudo-Classes (:) 
Describe a special state of an element. 
2.Pseudo-Elements (::) 
Allow you to style specific parts of an element. 

Syntax: 

selector:pseudo-class { 
   property: value; 
} 

selector::pseudo-element { 
   property: value; 
} 

Pseudo class selectors: 

:hover 
Styles element when mouse hovers over it. 
:active 
Styles element when it is being clicked. 
:visited 
Styles links after they are visited. 
:focus 
 Styles input when focused (clicked/tabbed). 
:checked 
Selects checked inputs. 
:first-child 
Selects the first child element inside a parent. 
:last-child 
Selects the last child element inside a parent. 
:nth-child(n) 
Selects nth child. 
:link  
 A pseudo-class that selects all unvisited links 
:nth-of-type(n)  
A pseudo-class that selects the nth element of its type (like the 2nd
