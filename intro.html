Implementation of CSS:

There are 3 ways to implement Css
  • Inline Css
  • Internal Css
  • External Css

1.Inline CSS:

We can write the Css code directly inside the HTML tag using style attribute. we can write in open tag. Syntax: h1 opentag style = ‚Äùcolor:blue;‚Äù close open tag Inline css h1 closing tag

Advantage: Quick and easy for single element.

Drawback: Not reusable, hard to maintain in big projects.

2.Internal CSS:

We can write the css code inside of style tag it should be placed inside of head tag.

Advantage: Styles multiple elements in the same page.

Drawback: Works only for that single HTML page (not reusable across many pages).

3.External CSS:

Written in a separate .css file and linked to HTML using link tag.

rel - relation

href - path of css file

Advantage: External CSS keeps code clean, reusable, consistent across pages, and improves performance with caching.

Drawback: Requires extra file load, can delay rendering, and if the file is missing the page looks unstyled.
